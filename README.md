# 📚 Global Best-Selling Books Data Analyzer

A high-fidelity data analytics application built with **Python**, **Pandas**, and **Streamlit** designed to digest, clean, and analyze historical literary market data. This analytical tool enables publisher-grade tracking of author performance indices, genre popularity shifts, and pricing correlation matrices.

## ✨ Core Features
* **Multi-Dimensional Query Filtering:** Features clean sidebar control nodes to segment datasets instantly by genre categories, pricing ranges, and year constraints.
* **Aggregated Statistical Computations:** Calculates rolling baseline metrics including mean pricing variables, total review accumulations, and author publication volume.
* **High-Contrast Tabular Reporting:** Renders clean, sorted data frame structures optimized for fast reading and presentation.
* **Minimalist UI Aesthetic:** Implements a professional, decluttered layout targeting analytical operations without unnecessary visual overhead.

## 🛠️ System Engineering Stack
* **Core Analytics Pipeline:** Pandas (Dataframe Operations & Lambda Mask Filtering)
* **Application Server UI:** Streamlit Engine
